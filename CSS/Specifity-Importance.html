There are rules for the specificity and the importance.

Let's check the object and value: Position when we have the same style

.li { color:red; color:blue; }

Color Blue will be applied here; The last term plays the importance when we have same objects in the styling.

Coming to the specificity:

li{color:blue};
.list-item{color:red};
li[draggable="true"] {color:purple};
#item{color:green};

Here color green will be applied importance increases in the downward way.

Types of CSS Styling will also matters here:

<. li el="stylesheet" href="styles.css" . />
<. style>internal styles; li { color : blue }
< li color : red> < /li .> : Inline Style

Here the importance is given to inline style then to internal styles and finally to external style sheets.

!important Keyword and it's importance

.item { color: blue; color:red !important; }

Here color:red will be applied. Applying !importance keyword to a property will make it the most important value to that relative property and element. "!important" keyword even dominatesd the inline styles and acts as the priority.
